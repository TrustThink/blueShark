## Research Software Disclaimer

BlueShark is an experimental research and educational project developed as part of a high school cybersecurity internship program.

This software is not intended for production use.

The code is provided for the purposes of:

- cybersecurity education
- research experimentation
- demonstrating automated PCAP analysis workflows

The software has not undergone formal security review, secure development lifecycle processes, or production hardening.

Users should assume that:

- the software may contain bugs
- security vulnerabilities may exist
- inputs such as malformed PCAP files may cause unexpected behavior

BlueShark should only be used in controlled research, educational, or testing environments.

It should not be deployed in production systems, security monitoring infrastructure, or environments containing sensitive data.
